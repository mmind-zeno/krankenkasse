# FKB Plus Zusatzversicherung

## Übersicht

Die FKB Plus Zusatzversicherung ist eine Ergänzung zur Grundversicherung und bietet erweiterte Leistungen in den Bereichen ambulante Behandlung, Medikamente, Spitalaufenthalt und mehr.

## Leistungen

| Leistung | Deckung |
| --- | --- |
| **Ambulante Behandlung** | Volle Deckung für Behandlungen durch Vertragsärzte und anerkannte medizinische Fachpersonen. |
| **Medikamente** | Von SWICA-anerkannten Ärzten und Therapeuten verordnete Präparate oder ärztlich verordnete Arzneimittel innerhalb der Indikation. |
| **Spitalaufenthalt** | Volle Deckung bei SWICA-Vertragsspitälern. |
| **Komplementärmedizin** | Bis CHF 80.–/Std. bei SWICA-anerkannten Ärzten und Therapeuten. |
| **Schwangerschaft** | CHF 200.– Stillgeld pro Kind. |
| **Prävention** | 90 % bis CHF 500.– pro 3 Jahre, gemäss separater Liste. |
| **Haushaltshilfe** | 50 % bis CHF 30.–/Tag, max. 60 Tage pro Jahr. |
| **Badekuren** | CHF 10.–/Tag, 21 Tage pro Jahr. |
| **Erholungskuren** | CHF 20.–/Tag, max. 30 Tage pro Jahr. |
| **Psychotherapie** | CHF 25.– pro Sitzung, max. 60 Sitzungen pro Jahr bei SWICA-anerkannten Psychotherapeuten. |
| **Brillen & Kontaktlinsen** | 90 % bis CHF 300.– pro 3 Jahre. |
| **Hilfsmittel** | 90 % bis CHF 500.– pro Jahr, gemäss separater Liste. |
| **Transporte** | Notfall-/Verlegungstransporte 90 % bis CHF 20 000.– pro Jahr, gültig für Schweiz und Ausland. |

## Ausschlüsse und Selbstbehalte

*   **Franchise:** Wahl zwischen CHF 0.– und CHF 600.– für Erwachsene.
*   **Selbstbehalt:** 10 % (max. CHF 700.– für Erwachsene bzw. CHF 350.– für Kinder).
*   Eine in der obligatorischen Krankenpflegeversicherung bereits erbrachte Kostenbeteiligung wird angerechnet.
