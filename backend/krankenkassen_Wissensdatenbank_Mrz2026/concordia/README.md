# Concordia Liechtenstein

Dieses Verzeichnis enthält die detaillierten Leistungsbeschreibungen der Zusatzversicherungen der Concordia.

## Produkte

*   **Ambulante Versicherungen:**
    *   [DIVERSA, DIVERSAplus, DIVERSAcare](./diversa.md): Umfassende ambulante Zusatzleistungen in drei Stufen.
    *   [NATURA, NATURAplus](./natura.md): Spezialisierte Deckung für Komplementärmedizin.

*   **Spitalversicherungen:**
    *   [Spitalversicherungen (ALLGEMEIN, HALBPRIVAT, PRIVAT, LIBERO)](./spital.md): Verschiedene Stufen für Komfort und freie Wahl im Spital.

*   **Spezialversicherungen:**
    *   [Zahnpflegeversicherung](./zahnpflege.md): Deckung für Zahnbehandlungen und Kieferorthopädie.
    *   [Ferien- und Reiseversicherung](./ferien_reise.md): Jahresversicherung für weltweiten Schutz auf Reisen.
